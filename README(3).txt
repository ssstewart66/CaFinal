 This is a company leave system based on Spring Boot framework, front-end using React.
Project tested on Java SDK 17 and Spring Boot@3.3.0
To run the project locally, add project to IDE and Maven will automatically handle the dependencies:

Spring Boot DevTools
Spring Web
Spring Session
Thymeleaf
JDBC API
Spring Data JPA
MySQL Driver
Validation
Java Mail Sender
CycloneDX SBOM support

Then connect to the local MySQL database by modifying login credentials in application.properties, and create a new schema called "javawebca".

After that, download NORD ISS and configure the corresponding environment variables.
cd to <project>\frontend directory, then
npm install axios react-router-dom react-csv

Start the project and run "npm start". In browser open localhost:8080, then go.
